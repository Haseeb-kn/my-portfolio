Non-Invasive Blood Glucose Monitor

This project focuses on developing a non-invasive blood glucose monitor using Near-Infrared (NIR) spectroscopy to measure glucose levels without the need for finger pricking. The system integrates ML modeling for accurate glucose level prediction, along with power management to ensure energy efficiency. The design includes PCB development, circuit testing, and precise calculations to optimize performance and reliability. This innovation aims to provide a painless, real-time monitoring solution for diabetic patients.

Smart Buck Converter with Voltage Mode Control

This project focuses on the design and implementation of a Buck Converter, a DC-DC step-down power converter, with Voltage Mode Control (VMC) for efficient voltage regulation. The system is modeled and simulated to optimize performance, ensuring stable output voltage under varying load conditions. PWM generation using TL494 is implemented to control the switching process. The design is tested in Proteus simulation, incorporating feedback mechanisms for enhanced stability and accuracy. Finally, the design is validated through hardware implementation, ensuring real-world applicability in power management systems.

RISC Processor Implementation on FPGA

This project involves the design and implementation of a RISC (Reduced Instruction Set Computing) processor using Xilinx, Quartus Prime, and ModelSim. The processor is designed to execute instructions efficiently with a simplified architecture, making it ideal for high-performance and low-power applications. The development process includes hardware description using HDL (Verilog/VHDL), simulation in ModelSim, and synthesis on FPGA platforms using Xilinx and Quartus Prime.

Design and Fabrication of Patch Antenna

The design process involves optimizing key parameters such as resonant frequency, gain, and radiation pattern to ensure efficient performance. After simulation and analysis in Ansys, the antenna is fabricated on a PCB board, followed by testing and validation to compare simulated and real-world performance. High-Gain Audio Amplifier for Condenser Microphone

This project designs a high-gain audio amplifier for a condenser microphone, ensuring a 100 Hz to 12 kHz frequency response with minimal distortion. It features a pre-amplifier using 2N3904 BJT and a class AB push-pull power stage with 2N3055 and MJ2955 transistors. Operating on a ±12V DC supply, it achieves an overall gain of 1000 V/V (60 dB) for efficient signal amplification

Single-Phase Transformer with Multiple Voltage Taps

This project focuses on the design and implementation of a single-phase transformer with a primary voltage of 12V and a secondary voltage of 24V while supplying 2A of current, yielding an apparent power of 48 VA. DC Variable Power Supply with Constant & Adjustable Voltage Output

This project focuses on designing a DC Variable Power Supply that provides both constant and adjustable voltage outputs. Using an LM317T voltage regulator, a zener diode, and a BJT 2N3904 transistor, the system ensures a stable 5V constant output while also allowing adjustable voltage regulation from a 24V step-down transformer. The project is essential for electronics testing, prototyping, and power management applications, ensuring smooth and noise-free voltage conversion.

Metal Detector using Mutual Inductance

Mutual inductance-based metal detectors rely on the interaction between two coils to detect metal objects, whereas self-inductance-based detectors utilize the changes in a single coil's magnetic field. The project provides a detailed theoretical analysis, circuit implementation, and practical evaluation of both methods, highlighting their advantages and limitations.

3D Shaft Modeling in AutoCAD

In this project, we design a 3D model of a mechanical shaft based on the given technical drawing using AutoCAD. The shaft consists of multiple stepped sections, keyways, and detailed features such as chamfers, fillets, and threaded areas. This project helps in understanding real-world engineering design, manufacturing requirements, and technical drawing interpretation for practical applications in mechanical engineering and CAD/CAM.
